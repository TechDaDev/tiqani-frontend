# Frontend Deployment

## Build

```bash
npm ci
npm run build
npm run start
```

## Required Environment

- `BACKEND_INTERNAL_URL`
- `NEXT_PUBLIC_SITE_URL`
- `NEXT_PUBLIC_APP_URL`
- `NEXT_PUBLIC_DEFAULT_LOCALE`
- `NEXT_PUBLIC_APP_NAME`

## Smoke

- homepage;
- login/logout;
- marketplace;
- contract detail;
- admin dashboard;
- notification feed;
- review flow;
- payment/dispute screens.
